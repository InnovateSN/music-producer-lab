# Data Processing & Privacy Summary (DPA Draft)

**Version:** 1.0

## 1. Roles
- Institution: Data Controller.
- Music Producer Lab: Data Processor.

## 2. Data Categories
- Account identifiers (name/email/role).
- Learning activity/progress data.
- Optional support ticket content.

## 3. Processing Purpose
- Service delivery, authentication, progress tracking, support, and security operations.

## 4. Subprocessors
A maintained subprocessor list must be provided before contract signature and updated on material changes.

## 5. International Transfers
Any cross-border transfer mechanism (e.g., SCCs) must be documented in executed DPA.

## 6. Retention & Deletion
- Retention windows defined by contract/SOW.
- Export/deletion workflow available at end of term.

## 7. Data Subject Rights
Processor supports Controller with access, correction, deletion, and portability requests.
