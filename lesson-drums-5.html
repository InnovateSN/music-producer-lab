Step 3 · Exercise

Program a fill in the 4th bar.

A drum fill in the last bar helps the loop transition when it restarts. Keep the K+S+HH groove from previous lessons for the first three bars, then create a short kick/snare fill in the final bar (steps 13–16).

For simplicity, we’ll use a 16-step grid that represents only the last bar (Beat 4, steps 13–16), but you should imagine programming the full pattern.

Target: open-ended. Focus on placing Kick and Snare hits in the last four steps (13, 14, 15, 16) so they feel like a brief break before the loop restarts at 1.
